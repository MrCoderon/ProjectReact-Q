# Project Q

### Live Demo: https://qcollins.herokuapp.com/

## Installation

Git clone this repository:

```
git clone https://github.com/qcollinscode/ProjectReactJS-Q.git
```

Change directory into the cloned repository:

```
cd ProjectReactJS-Q
```

Install the dependencies that are listed in the package.json file:

```
npm install
```

## Running

Start the application:

```
npm start
```
